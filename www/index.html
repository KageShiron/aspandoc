AsPandoc
Pandoc as a Service

Pandocを用いてオンライン上のドキュメントを別のドキュメントに変換するサービスです。

利用規定

  • 何ら保障はありません。自己責任でご利用ください。
  • バグや脆弱性のご報告はGitHubのissueまで
  • 負荷のかかる行為(直リンク、CI、ボットに使うなど)はやめてください。あまり真面目に運用する気はないので、不意に落ちたり仕様が変わります。
  • 攻撃しないでください。まじで。

Generic Options

from
(default: gfm) Pandoc Input Format.
See also pandoc's document.
to
(default: html) Pandoc Output Format.
See also pandoc's document.
stripyaml
(default:false) Remove yaml header in the document when true.

URL

Syntax:
/url?url={URL}

Example:
https://aspd.esora.xyz/url?url=https://github.com/KageShiron/aspandoc/blob/master/www/index.html&from=html&to=gfm

Gist

Syntax:
/gist/{id}

Example:
https://aspd.esora.xyz/gist/b59cdf3289f97567adc62e83419afb96?to=pdf

Esa.io

Convert markdown on esa.io to anothoer format.

Syntax:
/esa/{team}/{documentNumber}?token={access_token}

Example:
https://aspd.esora.xyz/esa/dev/102?token=XXXXXXXXXX&to=mediawiki

Note:

Using GET /v1/teams/:team_name/posts API.
By default, this application use the markdown source in json. However, it use body_html source when you specified from=html param.
